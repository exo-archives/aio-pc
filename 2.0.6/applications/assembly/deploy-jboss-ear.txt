1. Add an entry to the login-config.xml file located in the $JBOSS_HOME/server/default/conf directory with the following content:
-----------------------------------------
  <application-policy name="exo-domain">
	    <authentication>
	      <login-module code="org.exoplatform.frameworks.portletcontainer.access.StandaloneJAASLoginModule" flag="required">
	      </login-module>
	      <login-module code="org.exoplatform.frameworks.portletcontainer.access.StandaloneBroadcastJAASLoginModule" flag="required">
	      </login-module>
	    </authentication>
  </application-policy>
-----------------------------------------
